# BookVerse Project Context

## Project Name:
BookVerse

## Description:
BookVerse is a digital e-library and reading platform that helps users discover, read, manage, and share books online. It acts like a virtual library where readers can search books by title, author, or genre, access study materials, join reading communities, and track their reading journey.

## Features:
📚 Search and explore books

👨‍💻 Read e-books online

❤️ Save favorite books

🔍 Search by title, author, or genre

👥 Join public reading groups and discussions

📖 Personalized reading experience

☁️ Cloud-based digital library access

## Tech Stack:

Frontend: HTML, CSS, JavaScript

Backend: Node.js, Express.js

Database: MongoDB Atlas

API: Gutendex API

## Purpose:
To provide an easy digital library system where users can explore books and manage their reading list.

## Future Scope:
- AI book recommendation
- Social reading features
- Book reviews and ratings
